Project of Visual Studio 2019 Community.

Исходные коды проекта по приему и обработке сигналов спутниковых модемов вида VSAT-TES фирмы Hughes NS.
Суть: принимается и обрабатывается сигнал канала управления, и, на основе полученной из него информации, через COM-порты настраиваются модемы на соответствующие парные сигналы. 

Сигналы вводятся в ПЭВМ через интерфейсное устройство ввода  на базе Xilinx и TMS (далее драйвер), на вход которого поступают демодулированные сигналы  с модемов, далее - обработка и регистрация сигнала в соответствии с указанными ниже параметрами сигнала:
    QPSK-РВК16-НСК 3/4-V.35
    (фазовая модуляция QPSK-перемежитель РВК16 (цикл 16 бит)-сверточный декодер Витерби НСК-скремблер по рек.ITU-T V.35 (3,20)).
 
Общая инфа по сетям VSAT: https://www.osp.ru/nets/2000/04/141088 .
